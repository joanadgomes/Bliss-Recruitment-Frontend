# Bliss-Recruitment-Frontend

This is an application developed to Bliss Applications recruitment for a Junior Frontend position.

<br>

Library used: [React.js](https://react.dev/).


# Run the application

- Server at http://localhost:5173/.
- npm run dev on the terminal

# Setup

- Clone the repository
- Install the packages (npm install) on the terminal
